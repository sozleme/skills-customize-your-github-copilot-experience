
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman word-guessing game using Python, practicing string manipulation, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Initialize the Game

#### Description
Set up the game by randomly selecting a word and initializing the necessary variables for tracking game state.

#### Requirements
Completed program should:

- Randomly select a word from the predefined list
- Initialize variables for guessed letters, incorrect guesses, and maximum attempts
- Prepare the game for the main loop

### 🛠️ Implement Game Logic

#### Description
Create the main game loop that handles user guesses, updates the game state, and determines when the game ends.

#### Requirements
Completed program should:

- Display current progress with underscores for unguessed letters
- Accept letter guesses from the user
- Track incorrect guesses and remaining attempts
- Update the display after each guess
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win or lose messages
